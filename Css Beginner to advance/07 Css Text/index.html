text properties

Text Alignment:-

The `text-align` property controls the horizontal alignment of text within an element's box.

Text Decoration:-

The `text-decoration` property allows you to add decorations like underlines, overlines, or line-through to text.

Text Transform:-

The text-transform property allows you to control the capitalization of text.

Text Spacing:-

letter-spacing Property:- The letter-spacing property controls the spacing between characters in a text.

word-spacing Property:- The word-spacing property controls the spacing between words in a text.

The line-height-property is used to specify the space between lines

white-space is used to control how white space within an element is handled.

Text Shadow:-

The text-shadow property allows you to add a shadow effect to text characters.
